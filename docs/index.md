## Project Overview

A proof-of-concept agentic information-retrieval system that ingests documents (PDFs and web pages), stores them as vector embeddings in Qdrant, and exposes a query API backed by a LlamaIndex FunctionAgent. The agent autonomously reformulates queries, retrieves relevant context via hybrid search (dense + BM25), evaluates sufficiency, and synthesizes natural-language answers with source attribution.

## Quick Reference

- **Tech Stack:** Python · FastAPI · LlamaIndex · Qdrant · Gradio · Docker Compose
- **Entry Points:** `main.py` (server), `ingest.py` (CLI ingestion)
- **Architecture Pattern:** Agentic RAG pipeline (FunctionAgent → QueryEngineTool → hybrid vector search)
- **Database:** Qdrant (vector store, persistent Docker volume)
- **Deployment:** Docker Compose (app + qdrant services)

## Getting Started

### Prerequisites

- Python ≥ 3.10 (3.12 recommended)
- [uv](https://docs.astral.sh/uv/) package manager
- Docker & Docker Compose
- An API key: OpenAI **or** OpenCode Zen + HuggingFace embeddings

### Setup

```bash
cp .env.example .env   # configure API keys
uv sync                # install dependencies
docker compose up qdrant -d   # start vector DB
```

### Run Locally

```bash
uv run python main.py       # start API server on :8000
uv run python ingest.py     # ingest documents into Qdrant
```

### Run Tests

```bash
uv run pytest tests/
```
